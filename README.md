# Sales & Revenue Analysis Dashboard

## Project Overview

This project presents an interactive Power BI dashboard developed to analyze sales and revenue data. The dashboard provides insights into business performance through KPI tracking, trend analysis, regional comparisons, and product-level analysis.

## Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Sales Trend Analysis
* Category-wise Sales Analysis
* Region-wise Sales Analysis
* Top Performing Products
* Interactive Filters and Slicers

## Tools Used

* Power BI
* Microsoft Excel
* GitHub

## Files Included

* Sales_Revenue_Analysis_Dashboard.pbix
* Sales_Revenue_Dataset..xlsx
* Dashboard Screenshot

## Outcome

The dashboard helps users analyze sales performance, identify top-performing products, compare regional sales, and make data-driven business decisions through interactive visualizations.
